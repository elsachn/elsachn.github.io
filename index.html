Featured work

Research projects

Eight major AI + HVAC research projects completed during my Ph.D. research at NCUT — from prediction models to real-world deployed systems.

PROJECT / 01

Ice Storage AC — real-time monitoring

Developed a brine chiller power prediction model using DNN and an anomaly detection model using 1D convolutional autoencoder. Designed and deployed a real-time web-based monitoring application, embedded on Jetson Orin Nano.

DNN1D-CAEJetson Orin NanoWeb app
PROJECT / 02

Ice Rink Fogging Prevention

Predicted temperature and humidity of AHUs and ice rink environment. Applied clustering to analyze indoor/weather conditions and identified optimal operating parameters using a genetic algorithm to prevent fogging.

MLClusteringGenetic Algorithm
PROJECT / 03

Dual Chiller Performance Improvement

Predicted power consumption of chiller, fan, and pump. Applied clustering to analyze weather conditions, then optimized operating parameters, modes, and PLR to reduce total energy consumption.

MLClusteringPLR Optimization
PROJECT / 04

Data Center Cooling Optimization

Forecasted power consumption for IT and non-IT components. Developed a power prediction model for precision air conditioners using DNN, then applied a genetic algorithm to optimize AC settings for thermal control.

DNNLSTMGenetic Algorithm
PROJECT / 05

Axial Fan Design Optimization

Predicted fan pressure and airflow rate using DNN. Optimized design parameters with a genetic algorithm — combining deep learning with classical engineering design principles.

DNNGenetic AlgorithmFan Design
PROJECT / 06

Multi-Zone Building Optimization

Predicted supply air temperature of AHUs and chiller power consumption. Optimized operating parameters to maintain temperature comfort across multi-zone building environments.

MLAHU controlGenetic Algorithm
PROJECT / 07

Chiller Optimization with Data Mining

Built chiller power prediction models using neural networks. Applied clustering to analyze weather conditions and discovered hidden operational patterns using association rule mining.

Neural NetworksClusteringAssociation Rules
PROJECT / 08

Chiller Power Time-Series Forecasting

Developed chiller power prediction models using MLP. Built time-series forecasting models comparing MLP, 1D-CNN, and LSTM architectures for water-cooled chiller systems.

MLP1D-CNNLSTM
